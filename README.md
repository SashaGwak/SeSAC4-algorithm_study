SeSAC 알고리즘 스터디
=====================
## Devolopment environment
* programming Language
    * python
* Algorithm Site
    * Baekjoon Online Judge:https://www.acmicpc.net/

## 진행 방식 
* 문제를 풀고, 깃허브에 업로드
* 파일 이름 형식 
    * 백준 문제 : [BOJ]_문제번호_닉네임.확장자
* 커밋 메세지 형식 
    * 백준 문제 : [BOJ] 문제번호 문제이름 - 닉네임
* 문제를 풀며 공부한 내용, 책을 읽으며 공부한 내용을 디스코드에 공유
* 매주 월요일 11시 코드리뷰 및 문답식 회의 진행
